Data
=======

Welcome to PUNCH data!

Our latest version of data is 0j (learn more about versions `here <data_versions.rst>`_).
We will begin providing browse data and movies soon.

PUNCH is still being calibrated,
but you can take a look at the most up to date data
through the `Get Data page <https://punch.space.swri.edu/punch_science_getdata.php>`_
on our PUNCH website.
The most commonly used data products are L3_PAM and L3_CAM.

For more information about how to download data, see this `access page <data_access>`_.

For more information about data analysis tools, see the `examples here <auto_examples/index>`_.

For information on more data products, read this `introduction <data overview>`_.
